# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing
the maintainers directly. Do not open a public issue.

**Email:** [INSERT SECURITY EMAIL]

We will respond within 48 hours and work with you to understand and address the issue.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Security Best Practices

When using this learning platform:

- Run lessons in isolated workspaces
- Review code before executing shell commands
- Keep Neovim and system packages up to date
- Use the latest version of the CLI

Thank you for helping keep this project secure.
