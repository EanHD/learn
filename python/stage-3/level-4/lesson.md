# Level 4: Counting Problem

## Stage 3: Problem to Pseudocode

### Your Problem

Write a program that:

1. Asks user for a number
2. Prints all numbers from 1 to that number

Example: User enters 5

```
1
2
3
4
5
```

---

## ANSWER KEY

### Python Solution

```python
print("Enter a number:")
n = int(input())
for i in range(1, n + 1):
    print(i)
```
