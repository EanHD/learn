## Stage 3: Problem to Pseudocode

### Your Problem

Write a program that checks if someone can vote:

- If age >= 18: print "You can vote"
- Otherwise: print "You cannot vote yet"

---

## ANSWER KEY

### Python Solution

```python
print("What is your age?")
age = int(input())
if age >= 18:
    print("You can vote")
else:
    print("You cannot vote yet")
```
