# Interactive Task Manager

A simple command-line task management application that allows users to interactively add, view, complete, and delete tasks through a menu-driven interface.

## Features

- **Add Tasks**: Create new tasks with descriptions
- **View Tasks**: Display all tasks with completion status
- **Complete Tasks**: Mark tasks as completed
- **Delete Tasks**: Remove tasks from the list
- **Menu-Driven Interface**: Easy navigation with numbered options

## How to Run

```bash
python3 main.py
```

## Usage

1. Run the application
2. Choose from the menu options:
   - 1: Add a new task
   - 2: View all tasks
   - 3: Complete a task
   - 4: Delete a task
   - 5: Exit the application

## Example Interaction

```
         INTERACTIVE TASK MANAGER
```
