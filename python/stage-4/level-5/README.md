# Study Plan Advisor - Decision Support Application

An intelligent system that analyzes user preferences and goals to recommend personalized study plans. This decision support application uses a simple expert system approach to provide tailored learning recommendations.

## Features

- **Learning Style Assessment**: Determines preferred learning methods (reading, visual, hands-on, discussion)
- **Goal Analysis**: Identifies primary learning objectives
- **Time Assessment**: Evaluates available study time
- **Personalized Recommendations**: Generates customized study plans
- **Multiple Methods**: Suggests various study techniques based on preferences

## How to Run

```bash
python3 main.py
```

## Usage

1. Run the application
2. Answer the assessment questions about your learning preferences
3. Specify your goals and available time
4. Receive a personalized study plan recommendation

## Example Output

```
Focus: Skill-focused intensive practice

Recommended Schedule: 1-2 hour daily sessions with weekends off

Study Methods:
  1. Practical exercises
  2. Projects
  3. Experiments
  4. Solo deep work sessions

Additional Tips:
  1. Track your progress weekly
  2. Review material regularly for retention
  3. Take breaks to avoid burnout
  4. Adjust plan as needed based on progress
```
