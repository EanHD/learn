## Stage 1: Copying Code

### Today's Mission

Welcome to PowerShell! Today, you'll create your first PowerShell script. PowerShell is Microsoft's powerful command-line shell and scripting language for Windows administration and automation.

### Learning Goals

- Understand PowerShell script structure
- Learn how to execute PowerShell scripts
- Create your first script output
- Get comfortable with PowerShell syntax

### Your Task

Copy the following code EXACTLY as shown into a new file called `hello.ps1`:

```
Write-Host "Hello, World!"
```

### How to Execute

```
powershell -File hello.ps1
```

Expected output:

```
Hello, World!
```
