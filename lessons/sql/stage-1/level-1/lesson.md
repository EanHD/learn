## Stage 1: Copying Code

### Today's Mission

Welcome to SQL! Today, you'll create your first SQL query. SQL (Structured Query Language) is used to manage and query databases - a critical skill for any developer.

### Learning Goals

- Understand SQL query structure
- Learn how to use SQLite
- Create your first database query
- Get comfortable with SQL syntax

### Your Task

Copy the following code EXACTLY as shown into a new file called `hello.sql`:

```
SELECT 'Hello, World!' AS message;
```

### How to Execute

```
sqlite3 < hello.sql
```

Expected output:

```
Hello, World!
```
