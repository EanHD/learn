# Roadmap

This roadmap outlines planned features and improvements for the Learn project.

## v0.1.0 (Released - 2025-11-01)

- [x] Core CLI with interactive menu
- [x] C++ and Rust lesson support
- [x] Workspace scaffolding
- [x] Vim mode with stable split-screen
- [x] System verification (doctor)
- [x] First-time setup wizard (init)
- [x] Progress tracking

## v0.2.0 (Q1 2026) - Testing & Quality

### Planned Features

- [ ] **Lesson Autograder**: `learn test` to run automated tests
- [ ] **Lesson Submission**: `learn submit` to validate and mark complete
- [ ] **Test Framework**: Sample inputs/outputs for each lesson
- [ ] **CI/CD Pipeline**: GitHub Actions for linting and testing
- [ ] **Pre-commit Hooks**: Automated code quality checks
- [ ] **Enhanced Progress**: Detailed analytics and streaks

### Documentation

- [ ] MkDocs-based documentation site
- [ ] Video tutorials for Vim mode
- [ ] Troubleshooting guide by error message
- [ ] Glossary for beginners (LSP, Mason, etc.)

## v0.3.0 (Q2 2026) - Language Expansion

### Planned Languages

- [ ] Python (14+ lessons planned)
- [ ] JavaScript/TypeScript (12+ lessons)
- [ ] Go (10+ lessons)
- [ ] Lua (embedded scripting focus)

### Features

- [ ] Language-specific best practices guides
- [ ] Project-based capstone lessons
- [ ] Community-contributed lessons

## v0.4.0 (Q3 2026) - IDE & Cloud Integration

### Planned Features

- [ ] **VS Code Extension**: Native integration
- [ ] **Codespaces Support**: Zero-install cloud development
- [ ] **Dev Containers**: One-click environment setup
- [ ] **GitHub Classroom**: Integration for educators
- [ ] **Web-based Mode**: Learn in browser without install

### Accessibility

- [ ] High-contrast themes
- [ ] Screen reader compatibility
- [ ] Keyboard-only navigation
- [ ] Localization (i18n) framework

## v1.0.0 (Q4 2026) - Production Release

### Goals

- [ ] 500+ lessons across 6+ languages
- [ ] 10,000+ users
- [ ] Full test coverage
- [ ] Comprehensive documentation
- [ ] Active community contributions
- [ ] Partnerships with coding bootcamps

### Enterprise Features

- [ ] Team progress dashboards
- [ ] Custom lesson creation tools
- [ ] LMS integration (Canvas, Moodle)
- [ ] Analytics and reporting

## Future Considerations
