# Level 1: Hello World

## Stage 1: Copying Code

### Today's Mission

Welcome to Shell Scripting! Today, you'll create your first bash script. Shell scripts automate tasks on Unix/Linux systems and are essential for system administration and DevOps.

### Learning Goals

- Understand shell script structure
- Learn how to execute shell scripts
- Create your first script output
- Get comfortable with bash syntax

### Your Task

Copy the following code EXACTLY as shown into a new file called `hello.sh`:

```bash
#!/bin/bash
echo "Hello, World!"
```

### How to Execute

```bash
bash hello.sh
```

Expected output:

```
Hello, World!
```

### Success Checklist

- [ ] Created a file named `hello.sh`
- [ ] Copied the code exactly as shown
- [ ] Script executed without errors
- [ ] Saw "Hello, World!" printed

---

**Shell scripting is the glue that holds Unix/Linux systems together!**
