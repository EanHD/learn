### Today's Mission

Welcome to Swift! Today, you'll create your first Swift program. Swift is Apple's modern programming language used for iOS, macOS, watchOS, and tvOS development.

### Learning Goals

- Understand Swift program structure
- Learn how to run Swift code

### Your Task

Copy the following code EXACTLY as shown into a new file called `hello.swift`:

```swift

```

### How to Execute

```bash

```

Expected output:

```swift
Hello, World!
```
