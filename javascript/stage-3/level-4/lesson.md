## Stage 3: Problem to Pseudocode

### Your Problem

Print numbers from 1 to 10, each on a new line.

Example output:

```
1
2
3
...
10
```

---

## ANSWER KEY

```javascript
for (let i = 1; i <= 10; i++) {
    console.log(i);
}
```
