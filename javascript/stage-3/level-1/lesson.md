# Level 1: Greeting Problem

## Stage 3: Problem to Pseudocode

### Your Problem

Write a program that asks for a name and color, then prints a greeting combining them.

Example output:

```
Your greeting: Hello Sarah, your favorite color is blue!
```

---

## ANSWER KEY

```javascript
let name = "Sarah";
let color = "blue";

console.log("Your greeting: Hello " + name + ", your favorite color is " + color + "!");
```
