## Stage 3: Problem to Pseudocode

### Your Problem

Check if someone can vote (age 18 or older) and display the result.

Example output:

```
Age 20: Can vote!
Age 16: Too young to vote.
```

---

## ANSWER KEY

```javascript
let age1 = 20;
let age2 = 16;

if (age1 >= 18) {
    console.log("Age " + age1 + ": Can vote!");
} else {
    console.log("Age " + age1 + ": Too young to vote.");
}

if (age2 >= 18) {
    console.log("Age " + age2 + ": Can vote!");
} else {
    console.log("Age " + age2 + ": Too young to vote.");
}
```
