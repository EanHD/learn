## Stage 3: Problem to Pseudocode

### Your Problem

Print the multiplication table for 5 (5x1, 5x2, ... 5x10).

Example output:

```
5 * 1 = 5
5 * 2 = 10
...
5 * 10 = 50
```

---

## ANSWER KEY

```javascript
for (let i = 1; i <= 10; i++) {
    console.log("5 * " + i + " = " + (5 * i));
}
```
