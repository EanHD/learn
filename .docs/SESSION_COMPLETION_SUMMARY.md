## What You Can Now Do

### As a Learner

```bash
learn c++ 1           Open any lesson instantly
learn rust 3          Switch languages
learn --list          See all 70 available lessons
learn --info          Check your progress
learn --next          Get suggestion for next lesson

# Inside Vim:
<leader>n             Jump to next level
<leader>p             Go to previous level
C-l / C-h             Switch between lesson and code
:call NextLesson()    Smart navigation
```
